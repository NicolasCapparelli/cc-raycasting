2D Raycasting With 3D Scene Rendering

Using data from 2D Raycasting, we can create a 3D scene by taking the distance of each intersecting ray and using it as a grayscale value to paint one column of the canvas on the right side. By mapping that value to a height range, we can create the illusion of distance.
